# Description : 

    This project shows the collection of beers. Key features are:

        -> favorite features

# steps to run locally:

    ->  import to your desktop/ or download source code
    ->  move to location of your project  as '...\cheers-beers'
    ->  `npm install`
    ->  `npm run serve`
